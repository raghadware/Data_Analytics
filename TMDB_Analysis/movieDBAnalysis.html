Cleaning Data

  • drop unnecessary columns
  • drop duplicated rows
  • drop rows with null values
  • fix release_date data type from `str` to datetime
  • replace each `0` value with the column's mean in both `budget` and `revenue` columns

Conclusions

Result:

  • Adveture movies has the highest votes in the last couple years and it is the genre that costs more in production
  • High movie production budget can result in even higher revenues
  • Movie revenues in the last couple years reached around `64,966443`
  • Average movie production budget has declined from 1999 to 2015 with around `10,000000`
  • Top 5 most popular movies in the last 5 years: Jurassic World , Mad Max:Fury Road , Interstellar , Guardians of the Galaxy , Insurgent

Limitations:

  • the numbers for budget and revenue columns lack accuracy and some are missing
  • the data set contains multiple columns with multivalued columns which makes the analysis very limited.
  • data set do not provide movie data after 2015
  • there are a lot of NaN values.
